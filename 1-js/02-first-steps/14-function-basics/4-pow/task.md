# Funkcija pow(x,n)

Sukurkite funkciją `pow(x,n)`, kuri grąžina `x` iki `n` laipsnio. Kitaip tariant, padaugina `x` iš savęs `n` kartų ir grąžina rezultatą.

```js
pow(3, 2) = 3 * 3 = 9
pow(3, 3) = 3 * 3 * 3 = 27
pow(1, 100) = 1 * 1 * ...* 1 = 1
```

Sukurkite svetainę, kuri pateikia užklausas `x` ir `n` ir išveda rezultatą `pow(x,n)`.
